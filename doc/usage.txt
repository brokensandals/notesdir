usage: notesdir [-h] {mv,norm,t+,tc,t-,q} ...

optional arguments:
  -h, --help            show this help message and exit

Commands:
  {mv,norm,t+,tc,t-,q}
    mv                  move file and update references
    norm                normalize file
    t+                  add tags to files
    tc                  count number of files by tag
    t-                  remove tags from files
    q                   query for files
